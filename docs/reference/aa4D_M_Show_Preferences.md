# aa4D_M_Show_Preferences

## Overview

This page documents aa4D_M_Show_Preferences in the 4D_Info_Report reference.

## Description

The **aa4D_M_Show_Preferences** command without parameter shows on Disk the content of the folder «4D_Info_Report_Prefs».

If a pointer is passed as parameter, the path of the folder is returned via $1->. If the new preferences file «4DIR_General_Prefs.json» did not exist, it is created with the default values.
