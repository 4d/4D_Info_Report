# aa4D_NP_Report_Manage_Display

## Overview

This page documents aa4D_NP_Report_Manage_Display in the 4D_Info_Report reference.

## Description

The **aa4D_NP_Report_Manage_Display** command allows you to display a dialog to upload reports (or a selection of reports) from 4D Server, and copy them to a local folder.

You can also set (start and stop) via the dialog the stored procedure that creates reports on 4D Server. (Same as if executing aa4D_NP_Schedule_Reports_Server.)
