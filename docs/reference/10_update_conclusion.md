# Conclusion

## Overview

This page documents Conclusion in the 4D_Info_Report reference.

This component has already been widely used for more than 14 years, even with heavily loaded 4D Server. We advise installing it in your 4D Server application, so you can monitor the Server or create a report for Tech Support upon request without restarting your application.

A team at 4D will follow up on this project as I leave for retirement. This is why 4DInfoReport@4d.com is now mentioned in the component.
