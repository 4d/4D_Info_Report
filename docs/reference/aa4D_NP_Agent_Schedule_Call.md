# aa4D_NP_Agent_Schedule_Call

## Overview

This page documents aa4D_NP_Agent_Schedule_Call in the 4D_Info_Report reference.

## Description

The **aa4D_NP_Agent_Schedule_Call** command sets the number of seconds for the stored procedure that calculates values without creating reports, and returns a JSON answer.
