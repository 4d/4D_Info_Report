# aa4D_M_Folder_Rep_GetPath_local

## Overview

This page documents aa4D_M_Folder_Rep_GetPath_local in the 4D_Info_Report reference.

## Description

The **aa4D_M_Folder_Rep_GetPath_local** function lets you get the current location of «Folder_Reports».<br/>
(Its default location is next to the Data file. If the Folder_Reports did not exist, it will be created.)
