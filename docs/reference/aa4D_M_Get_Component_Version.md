# aa4D_M_Get_Component_Version

## Overview

This page documents aa4D_M_Get_Component_Version in the 4D_Info_Report reference.

## Description

The **aa4D_M_Get_Component_Version** function lets you get the current version of the component 4D_Info_Report_v4.

Example of content in the pointed variable, executing this shared method: `"4.96.1 20R10 (2026-01-13)"`

If the method is called without parameter, the version will be displayed in an ALERT.
