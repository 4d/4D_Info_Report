# aa4D_NP_Reports_Max_Get_Limit

## Overview

This page documents aa4D_NP_Reports_Max_Get_Limit in the 4D_Info_Report reference.

## Description

The **aa4D_NP_Reports_Max_Get_Limit** command retrieves the maximum number of last reports to be kept in the folder «Folder_reports» of the database, or 0 if no limit was set (default).
